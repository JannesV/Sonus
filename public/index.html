Welcome to Sonus! Draw in the white area to create sound. Based on the height the sound changes. You can change layers by pressing the buttons with the colored icons. Each layer contains a different sample-set. If you need some inspiration. Click on the view icon next to a user to copy his project to yours. You can adjust sliders per layer so you have full control over your sound.
